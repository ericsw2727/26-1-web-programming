게시글 가져오기

버튼을 누르면 API에서 게시글 데이터를 가져옵니다.

Request: 브라우저가 서버에 원하는 데이터나 작업을 요청하는 것

Response: 서버가 요청을 처리한 후 결과 데이터를 브라우저에 보내주는 것

API는 브라우저와 서버가 데이터를 주고받을 수 있게 해주는 창구 역할을 하기 때문에 필요

날씨 앱에서 현재 날씨 정보를 보여주려면 날씨 API에 요청해서 실시간 데이터를 받아와야 함

JSON은 서버와 브라우저가 데이터를 일정한 형식으로 쉽게 주고받기 위해 필요하다.

response.json(): 서버가 보낸 응답 데이터를 JSON 형식에서 자바스크립트 객체로 변환하는 역할을 한다.
